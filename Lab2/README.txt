Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, the user can use the keypad and enter numbers from 0-15. There are a total of 4 registers the numbers can be stored and will be saved in Hexadecimal. The user can then choose 2 register values to read as an input for an ALU composed of Multiplexers and Logic Gates. The ALU performs a bitwise left arithmetic shift of ALU input 2 by the amount in ALU input 1

-----------
FILES

-
Lab2.lgi

This file is the MMLogic schematic file. The file contains all the input switches, outputs and circuits for the lab to function.

-----------
INSTRUCTIONS

This program is intended to be run using Multimedia Logic (MML) software. Open the Lab2.lgi file in MML and click the green play button at the top to start the simulation.
